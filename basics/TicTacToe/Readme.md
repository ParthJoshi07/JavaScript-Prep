# Rock Paper Scissors Game

## Description
This is a web-based Rock Paper Scissors game built with HTML, CSS, and JavaScript. The player plays against the computer. The game features buttons for user input, dynamic result display, and simple animations.
## Features
- Interactive UI with buttons and animations
- Computer choice generated randomly
- Score counter for player and computer
- Responsive design with simple styling

## Files
- `index.html` - The main HTML page
- `styles.css` - CSS styles for the game interface
- `script.js` - JavaScript code containing game logic

## Requirements
- A modern web browser (Chrome, Firefox, Edge, Safari)
